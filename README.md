# PJT 소개 
- 운동 관련 영상 업로드 및 리뷰 작성 프로그램
- 개발 언어: Java, JSP, MySQL
- 조원: 이아영, 이재석, 이승민 

## 팀원 및 역할 분배
- 이아영: Review 기능 구현
- 이재석: Video 기능 구현
- 이승민: User 기능 구현 및 DB 생성
